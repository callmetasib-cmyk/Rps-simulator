# RPS Group Simulator

A browser-based visual rock-paper-scissors market simulator with 40 of each type. Entities bounce inside a box and convert defeated opponents on collision.

## Run locally

1. Install Node.js if needed.
2. Run `npm start`.
3. Open `http://localhost:3000` in your browser.

## How it works

- Start with 40 Rock, 40 Paper, and 40 Scissors.
- Each battle picks two random entities.
- If one beats the other, the loser converts into the winner's type.
- The simulation continues until only one type remains.
- The final surviving group wins.
